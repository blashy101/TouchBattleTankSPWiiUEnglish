# TouchBattleTankSPWiiUEnglish
Partial translation of Touch Battle Tank SP for Wii U

Game had mostly complete English localization images in the assets, took a couple assets to finish it from Nintendo Switch version of the game.

Note: When selecting option, Japanese text appears, it asks if you want to delete your save file. I tried changing this text to English but patching the Assembly dll either did nothing or crashed the game. 

Unzip archive to your MLC or SD folder where you have your games installed and if it prompts to overwrite, do so. Made using Wii U compatible UABE fork: https://github.com/blashy101/UABE-WiiU
